Markdown常见语法

标题

# 一级标题
## 二级标题
### 三级标题
其他以此类推

列表

无序列表：
只需要在文字前加上 -、+ 或 * 即可变为无序列表
例：

  • 1
  • 2
  • 3

有序列表：
直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格
例：

  1. 1
  2. 2
  3. 3

换行

两个或更多空格后回车就可以

引用

> 引用
效果：

引用

图片与链接

插入链接与插入图片的语法很像，区别在一个 !号
插入图片的地址需要图床，这里推荐 CloudApp 的服务，生成URL地址即可。
例：
[Baidu](http://baidu.com)
![Mou icon](http://mouapp.com/Mou_128.png)
效果：
Baidu

粗体与斜体

用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。
例：
**粗体
*斜体*
效果： 粗体* 斜体*

表格

暂时未实现？？
例：
| Item | Value | Qty |
| :——– | ——–:| :–: |
| Computer | 1600 USD | 5 |
| Phone | 12 USD | 12 |
| Pipe | 1 USD | 234 |

代码框

只需要用两个`把中间的代码包裹起来,使用 tab 键即可缩进
例：
`int i = 0;`
效果：
int i = 0;

分割线

分割线的语法只需要另起一行，连续输入三个星号 即可。
例：
\
效果：
