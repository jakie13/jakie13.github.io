ps:七牛云qshell命令行工具

使用七牛工具上传图片

初始配置

删除解压包内其他平台的工具，将可执行文件重命名为qshell
Windows 系统用户在 开始 菜单栏选择 运行 输入 cmd 回车即可打开 DOS 命令行窗口，然后切换到 qshell.exe 的所在磁盘路径。
设置 AccessKey 和 SecretKey
命令：qshell account AccessKey SecretKey

命令大全

  1. account 设置或显示当前用户的 AccessKey 和 SecretKey
  2. dircache 输出本地指定路径下所有的文件列表
  3. listbucket 列举七牛空间里面的所有文件
  4. prefop 查询七牛数据处理的结果
  5. fput 以文件表单的方式上传一个文件
  6. rput 以分片上传的方式上传一个文件
  7. qupload 同步数据到七牛空间，带同步进度信息和数据上传完整性检查
  8. qdownload 从七牛空间同步数据到本地，只支持同步某些前缀的文件和增量同步
  9. stat 查询七牛空间中一个文件的基本信息
  10. delete 删除七牛空间中的一个文件
  11. move 移动或重命名七牛空间中的一个文件
  12. copy 复制七牛空间中的一个文件
  13. chgm 修改七牛空间中文件的 MimeType
  14. fetch 从 Internet 上抓取一个资源到七牛空间中
  15. prefetch 更新七牛空间中从源站镜像过来的文件
  16. batchdelete 批量删除七牛空间中的文件，可以直接根据 listbucket 的结果来删除
  17. batchchgm 批量修改七牛空间中文件的 MimeType
  18. batchcopy 批量复制七牛空间中的文件到另一个空间
  19. batchmove 批量移动七牛空间中的文件到另一个空间 文档
  20. batchrename 批量重命名七牛空间中的文件 文档
  21. checkqrsync 检查 qrsync 的同步结果，主要通过比对 dircache 和 listbucket 的结果
  22. b64encode base64 编码工具，可选是否使用 UrlSafe 方式，默认 UrlSafe
  23. b64decode base64 解码工具，可选是否使用 UrlSafe 方式，默认 UrlSafe
  24. urlencode url 编码工具
  25. urldecode url 解码工具
  26. ts2d 将 timestamp（单位秒）转为 UTC+8:00 中国日期，主要用来检查上传策略的 deadline 参数
  27. tms2d 将 timestamp（单位毫秒）转为 UTC+8:00 中国日期
  28. tns2d 将 timestamp（单位 100 纳秒）转为 UTC+8:00 中国日期
  29. d2ts 将日期转为 timestamp（单位秒）
  30. ip 根据淘宝的公开 API 查询 ip 地址的地理位置
  31. qetag 根据七牛的 qetag 算法来计算文件的 hash
  32. unzip 解压 zip 文件，支持 UTF-8 编码和 GBK 编码
  33. privateurl 生成私有空间资源的访问外链
  34. saveas 实时处理的 saveas 链接快捷生成工具
  35. reqid 七牛自定义头部 X-Reqid 解码工具
  36. m3u8delete 根据流媒体播放列表文件删除七牛空间中的流媒体切片
  37. buckets 获取当前账号下所有的空间名称
  38. domains 获取指定空间的所有关联域名
