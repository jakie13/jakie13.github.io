使用hexo搭建个人博客

下载安装hexo

安装git

下载 msysgit 并执行即可完成安装。
地址：http://msysgit.github.io/

安装Node.js

在 Windows 环境下安装 Node.js 非常简单，仅须下载安装文件并执行即可完成安装。
地址：http://nodejs.org/

安装hexo

利用 npm 命令即可安装。（在任意位置点击鼠标右键，选择Git bash）
命令： npm install -g hexo

创建hexo文件夹

安装完成后，在你喜爱的文件夹下（如H:\hexo），执行以下指令(在H:\hexo内点击鼠标右键，选择Git bash)，Hexo 即会自动在目标文件夹建立网站所需要的所有文件。
执行命令： hexo init

安装依赖包

执行命令：npm install

本地查看

现在我们已经搭建起本地的hexo博客了，执行以下命令(在H:\hexo)，然后到浏览器输入localhost:4000看看。
执行命令：
hexo generate
hexo server

ps:Ctrl + C停止本地服务
初始界面：

关联github

创建页面仓库

这个仓库的名字需要和你的账号对应，格式: yourname.github.io
输入基本信息，然后点击创建仓库.

建立Repository

使用_config.yml文件，来建立关联.
命令：vim _config.yml
翻到最下面，改成我这样子的

ps:vim编辑后怎么退出 – esc后输入：号wq回车。具体可自行百度vim保存退出的几种方式
然后执行命令：
npm install hexo-deployer-git --save
然后，执行配置命令：
hexo deploy

部署步骤

每次部署的步骤，可按以下三步来进行。
hexo clean
hexo generate
hexo deploy

常用命令

hexo new"postName" #新建文章
hexo new page"pageName" #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

使用七牛工具上传图片

初始配置

删除解压包内其他平台的工具，将可执行文件重命名为qshell
Windows 系统用户在 开始 菜单栏选择 运行 输入 cmd 回车即可打开 DOS 命令行窗口，然后切换到 qshell.exe 的所在磁盘路径。
设置 AccessKey 和 SecretKey
命令：qshell account AccessKey SecretKey

命令大全

  1. account 设置或显示当前用户的 AccessKey 和 SecretKey
  2. dircache 输出本地指定路径下所有的文件列表
  3. listbucket 列举七牛空间里面的所有文件
  4. prefop 查询七牛数据处理的结果
  5. fput 以文件表单的方式上传一个文件
  6. rput 以分片上传的方式上传一个文件
  7. qupload 同步数据到七牛空间，带同步进度信息和数据上传完整性检查
  8. qdownload 从七牛空间同步数据到本地，只支持同步某些前缀的文件和增量同步
  9. stat 查询七牛空间中一个文件的基本信息
  10. delete 删除七牛空间中的一个文件
  11. move 移动或重命名七牛空间中的一个文件
  12. copy 复制七牛空间中的一个文件
  13. chgm 修改七牛空间中文件的 MimeType
  14. fetch 从 Internet 上抓取一个资源到七牛空间中
  15. prefetch 更新七牛空间中从源站镜像过来的文件
  16. batchdelete 批量删除七牛空间中的文件，可以直接根据 listbucket 的结果来删除
  17. batchchgm 批量修改七牛空间中文件的 MimeType
  18. batchcopy 批量复制七牛空间中的文件到另一个空间
  19. batchmove 批量移动七牛空间中的文件到另一个空间 文档
  20. batchrename 批量重命名七牛空间中的文件 文档
  21. checkqrsync 检查 qrsync 的同步结果，主要通过比对 dircache 和 listbucket 的结果
  22. b64encode base64 编码工具，可选是否使用 UrlSafe 方式，默认 UrlSafe
  23. b64decode base64 解码工具，可选是否使用 UrlSafe 方式，默认 UrlSafe
  24. urlencode url 编码工具
  25. urldecode url 解码工具
  26. ts2d 将 timestamp（单位秒）转为 UTC+8:00 中国日期，主要用来检查上传策略的 deadline 参数
  27. tms2d 将 timestamp（单位毫秒）转为 UTC+8:00 中国日期
  28. tns2d 将 timestamp（单位 100 纳秒）转为 UTC+8:00 中国日期
  29. d2ts 将日期转为 timestamp（单位秒）
  30. ip 根据淘宝的公开 API 查询 ip 地址的地理位置
  31. qetag 根据七牛的 qetag 算法来计算文件的 hash
  32. unzip 解压 zip 文件，支持 UTF-8 编码和 GBK 编码
  33. privateurl 生成私有空间资源的访问外链
  34. saveas 实时处理的 saveas 链接快捷生成工具
  35. reqid 七牛自定义头部 X-Reqid 解码工具
  36. m3u8delete 根据流媒体播放列表文件删除七牛空间中的流媒体切片
  37. buckets 获取当前账号下所有的空间名称
  38. domains 获取指定空间的所有关联域名

Markdown常见语法

标题

# 一级标题
## 二级标题
### 三级标题
其他以此类推

列表

无序列表：
只需要在文字前加上 -、+ 或 * 即可变为无序列表
例：

  • 1
  • 2
  • 3

有序列表：
直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格
例：

  1. 1
  2. 2
  3. 3

换行

两个或更多空格后回车就可以

引用

> 引用

引用

图片与链接

插入链接与插入图片的语法很像，区别在一个 !号
插入图片的地址需要图床，这里推荐 CloudApp 的服务，生成URL地址即可。
例：

粗体与斜体

用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。
例：
**粗体
*斜体*

sonar学习笔记

sonar简介

sonar是一款强大的代码质量管理软件。支持本地和服务端分析

sonarQube

sonarQube是sonar的服务端。使用sonarQube可以自己假设sonar的服务端，并通过网站访问管理。

下载

官网地址：http://www.sonarqube.org/
ps:

  1. 以前的版本是sonarQube和sonarRunner，现在已不支持。qube至少要5.X版本，最好使用最新版本
  2. sonarQube和scanner版本必须匹配，下载时可以看到对应的版本需求

mysql数据库配置

创建sonar数据库和用户
语句：
CREATE DATABASE sonar CHARACTER SET utf8 COLLATE utf8_general_ci;
CREATE USER 'sonar' IDENTIFIED BY 'sonar';
GRANT ALL ON sonar.* TO 'sonar'@'%' IDENTIFIED BY 'sonar';
GRANT ALL ON sonar.* TO 'sonar'@'localhost' IDENTIFIED BY 'sonar';
FLUSH PRIVILEGES;
查看运行结果：


ps：mysql至少需要5.6+版本，低版本不支持

配置sonarQube

配置文件为qube安装目录下的conf文件下的sonar.properties文件
主要配置内容：数据库地址、账号密码和数据库驱动
sonar.jdbc.username=数据库用户
sonar.jdbc.password=密码
sonar.jdbc.url=jdbc:mysql://localhost:3307/sonar? useUnicode=true&characterEncoding=utf8&rewriteBatchedStatements=true&useConfigs=maxPerformance
sonar.jdbc.driver:com.mysql.jdbc.Driver
ps:

  1. 数据地址后的四个设置必须带上，这是sonarQube的前置需求
  2. 数据库驱动只需要配置即可，sonarQube已经集成的大部分驱动，除了oracle。使用oracle数据库需要下载驱动到指定目录下
  3. 网上教程需要下载mysql驱动是因为版本低的原因（网上教程基本都是低版本的，只能作为参考）

R语言学习笔记（一）

什么是R语言？

参考：R语言实战（中文版）

是什么？

R是用于统计分析、绘图的语言和操作环境。R是属于GNU系统的一个自由、免费、源代码开放的软件，它是一个用于统计计算和统计制图的优秀工具。(百度)

为什么？

R语言具有非常多值得推荐的特性：

  1. 免费：多数商业统计软件价格不菲，投入成千上万美元都是可能的。而R是免费的！如果你是一位教师或一名学生，好处显而易见。
  2. 全面：R是一个全面的统计研究平台，提供了各式各样的数据分析技术。几乎任何类型的数据分析工作皆可在R中完成。
  3. 平台化：R是一个可进行交互式数据分析和探索的强大平台。
  4. 多数据支持：从多个数据源获取并将数据转化为可用的形式，可能是一个富有挑战性的议题。 R可以轻松地从各种类型的数据源导入数据，包括文本文件、数据库管理系统、统计软件，乃至专门的数据仓库。它同样可以将数据输出并写入到这些系统中。
  5. 扩展性：R是一个无与伦比的平台，在其上可使用一种简单而直接的方式编写新的统计方法。它易于扩展，并为快速编程实现新方法提供了一套十分自然的语言。
  6. 先进：R囊括了在其他软件中尚不可用的、先进的统计计算例程。事实上，新方法的更新速度是以周来计算的。如果你是一位SAS用户，想象一下每隔几天就获得一个新SAS过程的情景。
  7. 替代工具：如果你不想学习一门新的语言，有各式各样的GUI（Graphical User Interface，图形用户界面）工具通过菜单和对话框提供了与R语言同等的功能。
  8. 多平台：R可运行于多种平台之上，包括Windows、 UNIX和Mac OS X。这基本上意味着它可以运行于你所能拥有的任何计算机上。
